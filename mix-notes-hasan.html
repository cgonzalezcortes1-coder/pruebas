Cómo usar

Escribe o pega tus notas en el formato que ya usas. Un marker por línea, comenzando con el timecode.

01:10:20 Subir diálogo mamá 1.5 dB
1:23:45 Reverb vocals muy larga
23:10 Kick necesita más 80hz
02:14:30 Notas del director: quitar música

Acepta HH:MM:SS, H:MM:SS y MM:SS.
Las líneas sin timecode válido se ignoran.

En Pro Tools: File → Import → Session Data o arrastra el .txt a la ventana de Markers.
